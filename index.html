Instruction

Encription:

  1. Input in a text box the TEXT that want to Encrypt.

  2. Input in a text box the Encrypt Key.

  3. Select in a METHOD of what kind of encryption method to use.

  4. Click Encrypt button.

Dicryption:

  1. Input the Encrypt Key that you've inputed in the Decrypt Key text box.

  2. Click Decrypt button

Clear:

  1. Click Clear button to clear all the Encripted and Decrypted data.
